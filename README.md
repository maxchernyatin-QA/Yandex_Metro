В рамках проекта была поставлена цель протестировать мобильное приложение веб-приложения "Яндекс. Метро".

Выполнено:

1. Подготовлен чек-лист из 25 проверок, выполнено тестирование, обнаружено 10 багов.

2. Подготовлен чек-лист из 30 проверок, выполнено тестирование, обнаружен 1 баг.

3. Составлены 11 баг-репортов.

4. Составлен отчёт о проведённом тестировании и выявленных багах, сформулированы рекомендации по результатам тестирования.
